# Installation

## NPM (Recommended)

```bash
npm install -g quantwise
```

## System Requirements

- **macOS**, **Linux**, or **Windows** (WSL recommended)
- **Node.js** 18+ (only for npm install method)
- An **API key** — Anthropic (`sk-ant-...`), OpenAI-compatible, GLM, or any other supported provider

## Verify Installation

```bash
quantwise --version
```
