# Technical Analysis Skills

## /stock SYMBOL

Get real-time stock quotes and basic analysis.

```
/stock AAPL
/stock TSLA
/stock NVDA
```

- Real-time price data
- Key metrics and ratios
- Optional Notion storage integration

## /chart SYMBOL

Generate terminal candlestick charts with technical indicators.

```
/chart TSLA
/chart AAPL
```

- Terminal-rendered candlestick patterns
- Technical indicator overlays

## /technical-analyst

Comprehensive weekly chart technical analysis.

- **6-Step Methodology**:
  1. Trend analysis (uptrend/downtrend/sideways strength)
  2. Support and resistance identification
  3. Moving average analysis (20/50/200-week MAs)
  4. Volume analysis
  5. Chart pattern recognition
  6. Probabilistic scenario development (2–4 scenarios)
- **Output**: `[SYMBOL]_technical_analysis_[YYYY-MM-DD].md`
