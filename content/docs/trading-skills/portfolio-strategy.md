# Portfolio & Strategy Skills

## /portfolio-manager

Comprehensive portfolio analysis with Alpaca MCP integration.

- **Workflow**:
  1. Fetch portfolio data via Alpaca MCP
  2. Enrich with market data and fundamentals
  3. Asset allocation analysis
  4. Diversification assessment
  5. Risk analysis (beta, volatility, drawdown)
  6. Position-level analysis (top 10–15)
  7. Rebalancing recommendations
- **Output**: Markdown report with action items
- **Requires**: Alpaca MCP Server connection

## /weekly-trade-strategy

Multi-step workflow to generate a weekly trading strategy blog.

- **5 Stages**:
  1. Technical analysis (chart review)
  2. Market environment assessment
  3. News and catalyst analysis
  4. Blog generation (200–300 lines)
  5. Optional: Druckenmiller strategic overlay
- **Dependent Skills**: technical-analyst, breadth-chart-analyst, sector-analyst, market-environment-analysis, us-market-bubble-detector, market-news-analyst
- **Output**: `blogs/$DATE-weekly-strategy.md`
- **Time**: 15–30 minutes with data gathering

## /options-strategy-advisor

Options strategy recommendations and analysis.

- **Features**: Black-Scholes pricing, Greeks calculation, P/L simulation
- **Strategies**: Spreads, directional plays, hedges, income strategies

## /stanley-druckenmiller-investment

18-month strategic investment planning using Druckenmiller methodology.

- **Framework**: Macro-driven positioning
- **Scenarios**: Base, Bull, Bear, Tail risk
- **Focus**: High conviction trades with position sizing

## /institutional-flow-tracker

Track institutional investor flows via 13F filings.

- **Focus**: Large fund positioning changes, smart money signals
- **Data**: SEC 13F filings analysis

## /backtest-expert

Historical backtesting and strategy validation guidance.

- **Features**: Walk-forward analysis, robustness testing
- **Output**: Backtesting methodology and results interpretation

## /scenario-analyzer

Develop probabilistic market scenarios.

- **Output**: Base/bull/bear cases with probability weights

## /us-stock-analysis

Deep fundamental and technical analysis of individual US stocks.

- **Coverage**: Valuation, growth metrics, technical levels, risk/reward
